TIL about git shortlog

This feature is awesome, you can use it to easily see how many commits people are making to a repository. With the command 

git shortlog -ns

you are able to see who has committed the most to a repository. Here is the output of that command on the git repo;

These notes are meant to serve as a roadmap for people that come after me, and are looking to learn enough to be useful and contribute to git.

k&r

Understanding c thoroughly is important. Treat this as a sacred text.

baby-git

Understanding gits initial commit is something that you are able to do much easier than reading the current entire codebase.

pro git book

Read this so that you can start learning everything about git. Treat this as one of your sacred texts.

past git GSoC contributors

This is a great resource, it might make sense to review previous work done in order to familiarize oneself with the process.
